## 前端简介

- 前端框架采用单页面，需要多`tab`页建议使用浏览器的`tab`功能，会显示菜单标题。
- 文件均由`index.html`加载，子页面及其引用文件路径要相对于`index.html`文件。
- 子页面中所有变量都应该定义在`$(function(){这里面})`，避免覆盖全局变量。
- 部分组件使用前需要先调用组件提供的销毁方法，销毁旧的实例之后再创建新的实例。
