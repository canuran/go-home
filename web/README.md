## 前端项目

- 主要是上手门槛非常低，浏览器+常见代码编辑器即可开发，适合非专业的后端开发人员使用
- 采用单页面模式，所有文件都由index.html加载，所以子页面加载文件的路径也要相对于index.html
- 子页面中所有变量都应该定义在$(这里面)，防止覆盖全局变量，部分组件重复使用前可能要清理
